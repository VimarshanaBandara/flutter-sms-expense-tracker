# Flutter SMS Expense Tracker

## Features

- SMS parsing
- Expense / Income classification
- Auto categorization
- Transaction list
- Detail screen
- Category update

## Tech Stack

- Flutter
- Riverpod

## How to Run

flutter pub get
flutter run
